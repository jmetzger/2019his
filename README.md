## Settings in Eclipse 2019.09 vor a great presentation (Big Icons) 

Do this in the .eclipse.ini 

```
-Dswt.enable.autoScale=true
-Dswt.autoScale=200
-Dswt.autoScale.method=nearest
```

What does swt mean ?

SWT = The Standard Widget Toolkit 
https://www.eclipse.org/swt/
